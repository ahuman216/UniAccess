Demo Mode — Simulated Detections

Because a live Vision AI model requires a server-side API, VisionSense uses a simulation engine for this demo. Each detection randomly selects from a realistic set of objects and text commonly found in schools and public spaces.

In a deployed version, each captured frame would be sent to Tesseract.js (client-side OCR) or a cloud Vision API for real object and text recognition.
